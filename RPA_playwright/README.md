# 🤖 RPA Automation using Playwright (Python)

This folder contains example scripts demonstrating **Robotic Process Automation (RPA)** using Python and Playwright.
The scripts automate browser actions such as opening websites, interacting with elements, extracting information, and running automated tasks.

Playwright is a modern browser automation framework that supports **Chromium, Firefox, and WebKit** using a single API. ([GitHub][1])

# ⚙️ Installation

## 1️⃣ Clone the Repository

```
git clone https://github.com/Mks604/Gen_AI-Week_1.git
cd Gen_AI-Week_1/RPA_playwright
```

---

## 2️⃣ Create Virtual Environment (Recommended)

```
python -m venv venv
```

Activate environment:

### Windows

```
venv\Scripts\activate
```

### Mac / Linux

```
source venv/bin/activate
```

---

## 3️⃣ Install Dependencies

```
pip install playwright
```

Install Playwright browsers:

```
playwright install
```

---

# ▶️ Running the Scripts

Run any automation script using:

```
python script_name.py
```

Example:

```
python playwright_script_1.py
```

The script will launch a browser, perform automated actions, and close when the task is completed.

---

# 🧠 Example Automation Workflow

Typical RPA script steps:

1. Launch browser
2. Open a webpage
3. Locate elements
4. Perform actions (click, type, scroll)
5. Extract data
6. Close browser
