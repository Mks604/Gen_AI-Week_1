# 🐍 Python Concepts

This folder contains simple Python script examples demonstrating core Python fundamentals such as basic I/O, control flow, lists, dictionaries, loops, error handling, functions, and file handling. These are part of the Week 1 Python concepts exercises from the Gen AI course.

---

## 📁 Included Files

| File | Topic |
|------|-------|
| `calculator.py` | Simple calculator using input and arithmetic operators |
| `control_flow.py` | Control flow using `if`, `elif`, and `else` |
| `dictonary.py` | Dictionary usage examples |
| `error_handling.py` | Handling exceptions in Python |
| `file_handling.py` | Reading and writing files |
| `function.py` | Defining and using functions |
| `list.py` | Working with lists and indexing |
| `python_loops.py` | Using loops (`for`, `while`) in Python |

- **Functions** — Defining reusable code blocks

- **File I/O** — Reading and writing files
